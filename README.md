# Image Pixel Calculator

A lightweight and animated web app that calculates the number of pixels in an uploaded image.

## Features

- Stylish animated upload form.
- Cool burst animation after selecting an image.
- Reads image width and height in pixels.
- Calculates total pixels (`width Ã— height`).
- Shows megapixel value and image preview.

## Run

Open `index.html` in a browser.
